A professional version control tool for Elementor Global Styles. Capture, preview, and restore site-wide colors, typography, and custom CSS with ease.

== Description ==

Elementor Kit Backup Manager allows you to snapshot your Elementor Global Kit settings before making major design changes. 

Features include:
* One-click backup creation.
* Detailed preview of global colors, typography, and custom CSS before restoring.
* Version history storage for up to 20 versions.
* Import and export functionality to move styles between sites.
* User tracking to see who created each backup.
* RTL and localized support.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to 'Elementor' -> 'Kit Backups' to manage your versions.

== Frequently Asked Questions ==

= Does this backup the whole site? =
No, it specifically snapshots the "Global Kit" settings which include global colors, typography, and custom CSS defined in Elementor.

= Where are the backups stored? =
Backups are stored as an option in your WordPress database.
