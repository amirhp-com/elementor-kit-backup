# Elementor Kit Backup Manager

A lightweight, jQuery-based WordPress plugin to version-control Elementor Global Kit settings.

![Version](https://img.shields.io/badge/version-1.9.2-blue.svg)
![Author](https://img.shields.io/badge/author-AmirhpCom-indigo.svg)

## Features
- **Detailed Inspector**: Preview Colors, Typography settings, and Custom CSS directly from the backup without restoring.
- **One-Click Backup**: Snapshot your current Global Kit state (colors, typography, CSS).
- **History Management**: Stores up to 20 versions in the database.
- **Elementor Integration**: Conveniently located under the Elementor menu.
- **Import/Export**: Download snapshots as JSON or import them from other sites with integrity verification.
- **Custom UI Dialogs**: Clean Tailwind-based modals replace native browser alerts.
- **RTL Ready**: Full support for right-to-left languages.

## Installation
1. Upload the plugin to `/wp-content/plugins/`.
2. Activate it.
3. Access via **Elementor > Kit Backups**.
